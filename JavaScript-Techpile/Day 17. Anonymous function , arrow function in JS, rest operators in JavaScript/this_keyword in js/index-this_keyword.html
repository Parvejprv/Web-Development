This keyword in Java-Script

this keyword in java-script is used to get the current object of the element.

Here is the following example related to this keyword-

Calculator - Simple calculator

Note:- - Here, if we didn't pass any parameter to the function then it gives the undefined value to from it.

Calculator - Simple calculator (When we pass a specific value to the function)

Note:- - Here, if we pass any parameter to the function then it gives the that value that is passed into that

Calculator - Simple calculator (when we pass this keyword to the function - this will get the object)

Note:- - Here, if we pass this keyword as parameter to the function then it gives the current object.

Calculator - Simple calculator (when we pass this keyword to the function - this will get the object-value)

Note:- - Here, if we pass 'this.value' keyword as parameter to the function then it gives the current object-value.
