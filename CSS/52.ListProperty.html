There are two types of list:-
1. Ordered list
2. Un-ordered list


1. Ordered list
An ordered list uses the numbers and letters as the list item markers.
or
A list which has followed the order is known as "Ordered list".

  1. item 1
  2. item 2
  3. item 3
  4. item 4
  5. item 5
2. Un-ordered list
An unordered list uses bullets as the list item markers.
or
A list which has not followed the order is known as "Un-ordered list".


  • item 1
  • item 2
  • item 3
  • item 4
  • item 5
