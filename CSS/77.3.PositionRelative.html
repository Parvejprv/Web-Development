position property


position : static;

When we set the position property as static, then we can not place them at a particular place like top, right, left, bottom. Example:- h1 { position : static; border: 5px solid red; }

The given below code does not work-
Example:- h1 { position : static; border: 5px solid red; left: 50px; }


position : fixed;

When we set position for an element it always statys in the same place even if the page is scrollable. The top, right, bottom, left are used to position the element. Example:- img { height: 300px; width: 300px; border: 2px solid red; position: fixed; /* Here position property value set as fixed so that here we can position the element this element gonna placed or positioned at a particular place. */ right: 40px; top: 50px; } #outer { background-color: green; height: 300px; width: 300px; display: flex; justify-content: center; align-items: center; position: sticky; top: 0px; left: 0; } #main { background-color: blueviolet; height: 200px; width: 200px; display: flex; justify-content: center; align-items: center; }


position : relative;

When we set relative position for an element, elements positioned to its normal flow. Syntax:- selector_name { position : relative; top: 100px; right: 0px; border: 3px solid red; }


position : absolute;

When we set absolute position for an element, element is positioned to the nearest positioned ancestor. If there is no positioned ancestor then it follow the normal position according to browser. Syntax:- div { position : relative; width : 400px; height : 200px; border : 2px solid red; } div { position : absolute; top : 80px; right : 0; width : 200px; height : 100px; border : 3px solid red; }

Note:

We have seen similar two position value - relative & absolute then what is the difference between it?

The difference between these two are -
relative - this is positioned from its original position|place like 50px from top from its original position. absolute - this is positioned from it parent position|place.(absolute apne ancestor ke position ke hisab se khud ko positioned karta hai. It could be any of the position value like- fixed, relative, absolute etc. except static.)
