      /*  
      Grid-gap
      =========
      The gap between the grid item is known as grid-gap. this can be row-wise and column-wise both.

      - If it comes across the row then it is known as row-gap, and if it comes across the column then it is known as column-gap.
      


      grid-gap properties
      =====================
      - grid-row-gap
      - grid-column-gap
      - grid-gap




      - grid-row-gap
      ------------------
      If it comes across the row then it is known as row-gap.


      - grid-column-gap
      ------------------
      If it comes across the column then it is known as column-gap.


      - grid-gap
      ------------




      */



      .container {

        /* grid-row-gap: 2px; */

        /* grid-row-gap: 10%; */

        /* Let give the column-gap to the grid item */
        /* grid-column-gap: 1vw; */


        /* We can specify the given above these two property by just using the shorthand property grid-gap: gapValueOfRow gapValueOfColumn; */
